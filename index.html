はじめまして

はじめまして、shoです。
普通の会社員として働きながら、仙台を中心に活動しています。

ギター歴は7年程度で、現在は仙台にて
Guns N’ Rosesのカバーバンド 「Guns N' Loses」 にてリードギタリストとして活動中です。
洋楽・洋画が好きです。

バドミントンは中級で、社会人になってから復帰して4年ほどになります。

一時期は切り抜き動画を作成していたこともあり、ライブ映像やAI動画の作成経験もあります。

周りで仕事や学校での悩みを持つ人も多かったため、そういった方の力になれたらと思いサイトを立ち上げました。
自身も分からないことが出てくれば調べてお伝えすることもあると思います。

また、一人で引っ越しや荷物運び、買い物などで苦労される方もいると思い、そちらもお力添えできればと思っています。

丁寧で優しい対応を心がけますが、ロックの世界で生きてきたので、クレーマーやハラスメントは基本的に相手にしておりません。

できること

  • 機材購入相談（必要ならお店同伴も可能）
  • 弦交換サポート（別途料金）

⑤ 撮影（ライブ撮影・練習撮影など）

ライブ撮影だけでなく、バンド練習・ダンスグループなどの撮影も可能です。

撮影機材は Xperia 1 VII のみで行います（プロ機材ではありません）。
カメラは固定での撮影になります。

撮影データ保存用に MicroSDカード（SD）をご用意ください。
撮影後にSDへデータを移し、返却します。
※こちらでSDを用意することも可能です（SD代のみ頂きます）。

▶ MicroSDカードの詳細・価格目安は
こちらのページをご覧ください。

※炎天下の場合、スマホ温度上昇により撮影不可になる可能性があります。

⑥ 力仕事・買い物補助など

荷物運び、引っ越し補助、買い物補助など対応可能です。
仙台周辺だけでなく遠方もご相談ください。

※内容・時期・距離により料金は変動します。

⑦ ホームページ作成（GitHub Pages限定）

現在ご覧いただいているような、シンプルなホームページ作成も対応可能です。
GitHub Pages（無料）を利用した個人向けサイト限定になります。

料金表・プロフィール・お問い合わせ（LINE/メール）などを載せた
1ページ〜2ページ程度のホームページ作成が可能です。

※WordPress、独自ドメイン設定、予約システム、決済機能などの高度な機能は対応不可です。
※納品後の大幅な修正や継続的な更新作業は基本的に対応できません。
（簡単な修正は内容によって相談可）

料金（目安）

  • 相談（オンライン）：30分 3,000円 / 60分 4,500円
  • ギター指導（オンライン）：30分 3,000円 / 60分 4,500円
  • バドミントン指導（対面）：30分 3,000円 / 60分 4,500円（体育館代・交通費別）
  • 動画編集相談・VLLO指導：60分 4,500円
  • 撮影（ライブ・練習など）：30分程度 4,000円 / 60分程度 6,500円（交通費・入場料別）
  • 力仕事：内容により 3,500円〜（目安）
  • ホームページ作成（GitHub Pages / 1ページ）：10,000円〜
  • ホームページ作成（GitHub Pages / 2ページ）：15,000円〜

弦交換サポート：1,000円（弦代は別）

※対面の場合、交通費・施設費（スタジオ代、体育館代など）は別途になります。
※内容によっては追加料金が発生する場合があります。
